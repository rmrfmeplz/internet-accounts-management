# 互联网账号管理工具

一个**完全离线**、**隐私至上**的个人账号管家。专为解决个人账号数量过多、记不清账号归属与信息的核心痛点而设计。所有数据存储于本地，无需网络，零泄露风险，让你轻松掌控自己的账号信息。

## ✨ 核心特色

+ 🔒 纯本地运行：无需联网、无需服务器、无需数据库，启动即用
+ 📁 数据本地存储：所有账号信息以 JSON 文件存储于你的电脑中，永不外传
+ 🛡️ JWT 本地鉴权：即使本地运行，也提供接口访问保护，防止未授权操作
+ 🖥️ 桌面端体验：基于 Electron 封装，提供独立窗口，操作更专注
+ 🚀 轻量极速：Vue3 + Node.js 极简技术栈，启动快、响应快

## 🧰 技术栈

| 模块   | 技术           | 用途                |
|------|--------------|-------------------|
| 前端   | Vue3         | 用户界面与交互逻辑         |
| 后端   | Node.js      | 本地 API 服务、文件操作    |
| 存储   | JSON 文件      | 替代数据库，轻量持久化       |
| 桌面化  | Electron     | 将 Web 应用打包为桌面端    |
| 进程管理 | concurrently | 同时启动前后端及 Electron |
| 身份验证 | jsonwebtoken | 本地接口权限校验          |

## 🗂️ 项目结构

本项目采用前后端分离 + 桌面端封装的目录结构，各模块解耦，便于单独开发和维护：

```text
internet-accounts-management/ # 项目根目录
├── backend/                  # Node.js 后端服务（API + JSON 存储 + JWT）
├── frontend/                 # Vue3 前端界面（可视化账号管理）
├── electron/                 # Electron 桌面应用封装
├── package.json              # 项目主配置（一键安装/启动脚本）
└── README.md                 # 项目说明文档
```

## 🚦 快速开始

### 环境准备

+ 已安装 Node.js（包含 npm）

### 第一步：获取代码

```text
git clone https://github.com/rmrfmeplz/internet-accounts-management.git
cd internet-accounts-management
```

### 第二步：配置 JWT 密钥（重要！）

进入后端目录：
`cd backend`

创建`.env`文件，并填入以下内容：

`JWT_SECRET=你的自定义强密钥`

💡 密钥建议：长度大于 16 位，混合大小写字母、数字及符号。

### 第三步：一键安装依赖

返回项目根目录，执行：

`npm run install:all`

该命令会自动安装根目录、backend、frontend、electron 下的所有依赖。

### 第四步：一键启动

`npm start`

### 停止应用

在启动应用的终端窗口中，按 Ctrl + C。

## 💾 数据存储说明

1. 所有数据均存储在`backend/data/`目录下的 JSON 文件中。
2. 前端不直接操作文件，所有读写均通过后端 API 进行。
3. **重要提醒**：JSON 文件是你数据的唯一存储位置，请定期备份，避免误删或系统重装导致数据丢失。

## ⚠️ 注意事项

1. **单用户设计**：本工具为个人单机使用设计，不支持多用户同时操作。
2. **密钥安全**：`.env`文件中的`JWT_SECRET`务必保密，切勿提交至公开仓库。
3. **安全提醒**：本项目仅管理账号基本信息，不存储密码等敏感信息，请使用专业密码管理器保管密码。
